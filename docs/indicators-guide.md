## 1. 核心哲学

### 1.1 什么是"持仓结构"？

期货市场的龙虎榜（每日席位持仓排名）不仅告诉你**谁持有多少**，更重要的是告诉你**持仓的分布形态**——这就是"结构"。

一个健康的趋势通常伴随：
- 主力席位集中度稳定或上升（**收紧**）
- 换手率正常，没有大规模调仓
- 前排席位稳定，没有频繁进出
- 多空力量对比有明确方向

而当结构开始**松动**时，往往是趋势转折的早期信号：
- 集中度从高位回落——主力在派发
- 换手率突然飙升——资金在剧烈博弈
- 大量新席位进入前排——原有格局被稀释
- 核心席位翻仓——聪明钱改变了方向

### 1.2 系统设计的核心理念

```
持仓数据（每日龙虎榜）
        │
        ▼
   ┌─────────────────────────┐
   │  P0: 宏观结构层          │  CR 趋势、集中度演变
   │  P1: 席位行为层          │  叛变检测、聪明钱追踪
   │  P2: 微观结构层          │  换手率、新进入者、分歧度
   │  P3: 综合评分层          │  结构健康度、全品种对比、预警
   └─────────────────────────┘
        │
        ▼
   交易决策参考
```

**每一层回答一个不同的问题**：
- P0：持仓结构整体在收紧还是松动？
- P1：具体是哪些席位在搞事情？
- P2：有没有更深层的微观结构变化？
- P3：综合来看，这个品种现在能不能碰？

## 3. 面板详解

### 3.1 CR 趋势图

**位置**：📈 持仓结构演变 → 📈 CR 趋势图

**包含子图**：
- **CR 时间序列**（上）：CR 多头（绿线）和 CR 空头（红线）随时间变化
- **HHI 时间序列**（左下）：Herfindahl-Hirschman 指数，比 CR 更全面的集中度度量
- **总持仓量**（右下）：多头总持仓和空头总持仓的柱状对比

#### CR（集中度比率，Concentration Ratio）

```
CR_long  = Top-3 多头持仓之和 / 全部多头持仓之和
CR_short = Top-3 空头持仓之和 / 全部空头持仓之和
```

| CR 值 | 含义 |
|-------|------|
| > 40% | 高度集中——Top-3 席位掌握定价权 |
| 20–40% | 中等集中——正常市场状态 |
| < 20% | 高度分散——缺乏主导力量 |

**关键用法**：
- CR 长期 > 40% 且持续上升 → 主力在集中筹码，趋势可能延续
- CR 从高位回落 → **结构松动信号**，主力可能在出货
- CR 多头与 CR 空头交叉 → 力量对比发生根本性转变

#### HHI（赫芬达尔指数）

```
HHI = Σ(share_i²)   对所有 i 个席位
```

| HHI 值 | 含义 |
|--------|------|
| > 0.25 | 极高集中（接近垄断） |
| 0.10–0.25 | 中等集中 |
| < 0.10 | 高度分散 |

HHI 比 CR 更敏感——它考虑**所有席位**的分布，而不仅仅是 Top-3。当 HHI 与 CR 出现背离时（一个在涨一个在跌），说明中小席位正在发生结构性变化，而 Top-3 尚未反映。

#### 图表解读示例

```
CR 多头 ████████░░░░  0.42  (上升中 → 收紧)
CR 空头 ██████░░░░░░  0.31  (下降中 → 松动)
─────────────────────────────────────
→ 多头集中度在收紧，空头集中度在松动
→ 多空差在扩大（0.42 - 0.31 = 0.11）
→ 偏多格局确立
```

### 3.2 全景评分

**位置**：📈 持仓结构演变 → 📊 全景

**包含**：
- **综合评分卡**（顶部）：6 维度评分 + 总分 + 状态标签
- **换手率 & 新进入者**（上行）：两个结构松动的前瞻指标
- **多空博弈 & 集中度剖面**（下行）：粒度更细的结构分析

#### 综合结构健康度评分（0–100）

| 分数 | 状态 | 含义 | 交易建议 |
|------|------|------|---------|
| 80–100 | 🟢 健康 | 结构稳定，无松动信号 | 可按偏见方向正常交易 |
| 60–79 | 🟡 关注 | 部分指标出现松动迹象 | 减仓或收紧止损 |
| 40–59 | 🟠 松动 | 多指标共振松动 | 建议观望或轻仓 |
| 0–39 | 🔴 危险 | 结构剧烈变动 | 停止交易该品种 |

**六个评分维度**：

| 维度 | 满分 | 评的是什么 |
|------|------|-----------|
| CR 稳定性 | 25 | 集中度趋势是否稳定？松动/收紧都会扣分 |
| 换手率 | 20 | 持仓变动比例。低换手=稳定，高换手=剧烈调仓 |
| 新进入者 | 15 | Top-20 席位中有多少新面孔？新面孔多=格局在变 |
| 分歧度 | 15 | 多空力量对比是否在变化？收敛或发散都会扣分 |
| 叛变 | 15 | 近期有多少席位发生了方向翻转或大幅减仓？ |
| 排名稳定 | 10 | Top-3 席位阵容是否稳定？频繁换人=格局不稳 |

**实际案例**（铁矿石 2026-07-10）：
```
总分: 90/100 🟢 健康
├── CR 稳定性   25/25  集中度趋势稳定
├── 换手率      20/20  低换手(5.2%)
├── 新进入者    15/15  无新进入者
├── 分歧度      15/15  力量对比稳定
├── 叛变         5/15  3起(高危0)    ← 扣分项
└── 排名稳定    10/10  高度稳定(100%)
```

#### 换手率（Position Turnover Rate）

```
换手率 = Σ|long_change_i| / Σ(long_pos_i)
```

衡量持仓的"活跃程度"。注意这里用的是 `long_change`（日增减仓量），不是成交量。

| 换手率 | 含义 |
|--------|------|
| < 15% | 持仓稳定，席位以持有为主 |
| 15–30% | 正常换手 |
| 30–50% | 偏高，资金在调仓 |
| > 50% | 剧烈换手，结构可能即将变化 |

**关键信号**：
- 换手率突然从 10% 飙升到 40% → **强烈预警**，有大资金在进出
- 换手率持续走高但价格不动 → 多空激烈博弈，即将突破

#### 新进入者比率

```
新进入者比率 = 今天Top-20中新面孔数 / 20
```

| 比率 | 含义 |
|------|------|
| 0% | 前排席位稳定 |
| < 15% | 少量新进入者——正常 |
| 15–30% | 较多新面孔——结构在稀释 |
| > 30% | 大量换血——格局剧烈变动 |

**为什么重要**：新资金入场通常会打破原有的力量平衡。当之前不在 Top-20 的席位突然大量进入时，说明市场吸引了新的主力资金，原有的集中度结构可能被稀释。

#### 多空分歧度

```
分歧度 = 1 − |CR_long − CR_short|
```

| 值 | 含义 |
|----|------|
| 接近 1.0 | 多空近乎平衡，没有一方占优 |
| 接近 0.5 | 一方明显占优 |
| 接近 0 | 极端倾斜，一方碾压 |

趋势方向更重要：

| 趋势 | 含义 |
|------|------|
| **收敛**（converging） | 多空差在缩小→力量对比在变化→可能即将反转 |
| **发散**（diverging） | 多空差在扩大→一方正在加强主导 |
| **稳定**（stable） | 力量对比维持不变 |

#### 集中度剖面

展示不同级别（CR1=第一大, CR3=前三大, CR5=前五大, CR10=前十大）的集中度曲线。

```
CR1_long  = 10.6%    ← 最大的多头席位只占 10.6%
CR3_long  = 31.0%    ← 前三大占 31%
CR5_long  = 47.8%    ← 前五大占近一半
CR10_long = 71.9%    ← 前十大占 72%
```

同时计算 **Gini 系数**——如果 Gini 接近 0 说明持仓均匀分布，接近 1 说明极端集中。

**关键信号**：
- CR1 很低（< 10%）但 CR10 很高（> 70%）→ 没有绝对的龙头，但前十名集体控盘
- CR1 很高（> 20%）→ 一家独大，容易操纵价格

### 3.3 结构事件

**位置**：📈 持仓结构演变 → 🚨 结构事件

系统自动检测 5 种结构事件：

| 事件类型 | 触发条件 | 含义 |
|---------|---------|------|
| **long_loosening** | CR 多头从 tighten→loosen | 多头集中度从收紧变为松动 |
| **short_loosening** | CR 空头从 tighten→loosen | 空头集中度从收紧变为松动 |
| **cr_cross** | CR_long 与 CR_short 交叉 | 多空力量对比翻转 |
| **hhi_divergence** | HHI 与 CR 走势背离 | 中小席位在暗中变化 |
| **position_surge** | 总持仓单日变化 > 20% | 大资金进出 |

**事件严重度**：
- 🔴 **high**：确认的趋势转变或关键交叉
- 🟡 **medium**：值得注意的变化，但尚未确认
- 🟢 **low**：轻微变化，参考即可

### 3.4 席位排名变动

**位置**：📈 持仓结构演变 → 🔍 席位排名变动

#### Jaccard 稳定性指数

```
Jaccard 稳定性 = |昨天Top-3 ∩ 今天Top-3| / |昨天Top-3 ∪ 今天Top-3|
```

| 值 | 含义 |
|----|------|
| 1.0 | Top-3 完全没变 |
| 0.67 | 3 个中换了 1 个 |
| 0.33 | 换了 2 个 |
| 0.0 | Top-3 全部换人 |

**震荡阈值**：Jaccard < 0.5 标记为 "shakeup"。

**为什么重要**：Top-3 席位换人通常意味着：
- 原来的主力在撤退
- 新的主力在进场
- 市场的"控盘方"发生了变化

### 3.5 席位叛变

**位置**：📈 持仓结构演变 → 🚨 席位叛变

#### 叛变日志（主视图）

按日期从新到旧排列，每天的事件一目了然——类似 Git log。

#### 5 种叛变类型

| 类型 | 触发条件 | 严重度 | 交易含义 |
|------|---------|--------|---------|
| 🔴 **多翻空** | 净持仓从正变负 | high | 多头主力投降，最强烈的看空信号 |
| 🟢 **空翻多** | 净持仓从负变正 | high | 空头主力投降，最强烈的看多信号 |
| 🟠 **多头减仓** | 净多头减少 >30% | high/med | 多头在撤离，结构松动 |
| 🟠 **空头减仓** | 净空头减少 >30% | high/med | 空头在回补，结构松动 |
| 🟡 **动量逆转** | 连续加仓方向反转 | medium | 此前的惯性行为被打破 |

#### 叛变时间轴气泡图（辅助视图）

- **横轴** = 日期
- **纵轴** = 席位名
- **气泡大小** = 严重度 × 仓位变化量（high severity 气泡更大）
- **气泡颜色** = 叛变类型

Hover 悬停可查看完整的事件描述和指标。

#### 席位 × 叛变类型矩阵

一张交叉表，行是席位、列是 5 种叛变类型。席位按"涉及类型数"排序——叛变类型最多的排最前面。

**用法**：如果一个席位同时出现在多种叛变类型中（比如既多翻空又大幅减仓），说明这个席位的方向转变非常坚决。

### 3.6 聪明钱

**位置**：📈 持仓结构演变 → 💰 聪明钱

#### 工作原理

对每个席位，计算其**净持仓变动（ΔNP）**与**未来 N 日价格涨跌幅**之间的相关性：

```
ΔNP(t) = (long(t) - short(t)) - (long(t-1) - short(t-1))
price_change(t+N) = close(t+N) / close(t) - 1

Pearson r = Corr(ΔNP(t), price_change(t+N))
```

测试 N = 1, 3, 5 天，取 |r| 最大的 lag 作为该席位的"最佳领先天数"。

#### 指标说明

| 指标 | 含义 |
|------|------|
| **Pearson r** | 线性相关系数。> 0.3 视为显著 |
| **Spearman ρ** | 秩相关系数，更抗异常值 |
| **命中率** | ΔNP 方向与未来价格方向一致的概率 |
| **平均影响** | 加多日后平均涨幅 − 加空日后平均涨幅 |
| **领先天数** | 该席位仓位变化领先价格变化的天数 |

#### 席位分类

| 类别 | 含义 | 怎么用 |
|------|------|--------|
| 🟢 **领先多头** | 加多 → 后续价格上涨 | 跟随信号，席位加多时看多 |
| 🔴 **反向指标** | 加多 → 后续价格下跌 | 逆向信号，席位加多时反而谨慎 |
| ⚪ **噪音** | 无明显预测力 | 该席位的仓位变化不具参考价值 |

#### 散点验证

- **单个席位散点**：每个 broker 卡片内的 expander，展示 ΔNP(x) vs 未来收益(y) 的散点图 + 回归线
- **散点网格**：📈 散点验证 tab，前 6 个席位以子图网格展示，可一眼对比

**判断技巧**：
- 散点呈明显的左上→右下或左下→右上分布 → 相关性真实
- 散点是一个或两个极端值拉动的 → 相关性可能是假象
- 散点完全无规律 → 该席位确实无预测力

## 5. 席位偏见总览

**位置**：📊 席位偏见总览（全部品种）

这是最简洁的每日交易参考：

| 字段 | 含义 |
|------|------|
| **偏见方向** | ▲看多 / ▼看空 / ◆中性 / ◇跳过 |
| **确信度** | 0–100%，信号有多强 |
| **多头集中度** | Top-3 多头的持仓占比 |
| **空头集中度** | Top-3 空头的持仓占比 |
| **ΔNP** | 净持仓动量，正=加多，负=加空 |
| **加多/加空席位** | 今日 ΔNP 最大的 3 个席位 |

**偏见判定规则**：

| 条件 | 偏见 |
|------|------|
| CR 多头 > 40% 且 > CR 空头 且 ΔNP > 0 | ▲ 看多 |
| CR 空头 > 40% 且 > CR 多头 且 ΔNP < 0 | ▼ 看空 |
| 多空集中度均 > 40% | ◆ 中性（主力对决） |
| 多空集中度均 < 20% | ◇ 跳过（无合力） |
| 其他 | ◆ 中性 |

## 6. 指标词汇表

| 缩写/术语 | 全称 | 公式 |
|-----------|------|------|
| **CR** | Concentration Ratio（集中度比率） | Top-N 持仓之和 / 总持仓 |
| **HHI** | Herfindahl-Hirschman Index | Σ(share_i²) |
| **ΔNP** | Net Position Momentum（净持仓动量） | ΔNP = NetPos(t) − NetPos(t-1) |
| **换手率** | Position Turnover Rate | Σ\|long_change\| / Σ long_pos |
| **分歧度** | Divergence Index | 1 − \|CR_long − CR_short\| |
| **Gini** | Gini Coefficient | 持仓份额的洛伦兹曲线面积比 |
| **Jaccard** | Jaccard Similarity | \|A ∩ B\| / \|A ∪ B\| |
| **Pearson r** | Pearson Correlation | Cov(x,y) / (σ_x · σ_y) |
| **Spearman ρ** | Spearman Rank Correlation | 秩变量的 Pearson r |
| **命中率** | Hit Rate | sign(ΔNP) = sign(收益) 的比例 |

## 7. 实战使用指南

### 7.1 日常使用流程

```
盘前（9:00 前）
├── 1. 打开 Dashboard
├── 2. 查看 🌡 全品种热力图 → 排除红色/橙色品种

    ├── 📊 全景 → 看综合评分，低于 60 考虑放弃
    ├── 🚨 席位叛变 → 有没有关键席位翻仓？
    └── 💰 聪明钱 → 领先多头的席位今天在做什么？

盘中
├── 5. 关注 K 线图中的 Stop Run 标记
├── 6. 关注 🚨 信号推送（Stop Run + 偏见共振）
└── 7. 如果结构评分骤降 → 立即减仓

盘后（15:00 后）
├── 8. 运行 python src/fetcher_akshare.py --date YYYYMMDD
├── 9. 刷新 Dashboard
└── 10. 回顾 📈 CR 趋势图 → 今天的结构变化
```

### 7.2 信号分级

| 级别 | 条件 | 行动 |
|------|------|------|
| ⭐⭐⭐ | 偏见方向 + 结构健康 + Stop Run | 正常仓位 |
| ⭐⭐ | 偏见方向 + 结构关注 + Stop Run | 半仓 |
| ⭐ | 偏见方向 + 结构松动 | 观望或 1/4 仓 |
| ❌ | 结构危险 或 偏见=跳过 | 不交易 |

### 7.3 常见场景

**场景 1：CR 多头在高位，但叛变面板显示 Top-3 多头中的 2 个在减仓**

→ 结构松动预警。虽然整体 CR 还没降，但核心席位已经开始撤退。考虑减仓。

**场景 2：换手率突然从 10% 飙升到 45%，但价格没动**

→ 多空激烈博弈。可能即将出现大行情。注意叛变面板中谁在翻仓，以及聪明钱面板中谁的命中率高。

**场景 3：综合评分 55（松动），但偏见是 bearish**

→ 评分低说明结构不稳定，不建议重仓。但如果多指标共振偏向同一方向（如 CR 空头 tighten + 多翻空事件 + 高命中率席位加空），可以轻仓跟随。

**场景 4：聪明钱 panel 显示某席位 r=0.85，但该席位今天是噪音（|r|<0.3）**

→ 可能是相关性计算窗口太短或数据噪音。查看散点图验证——如果散点呈明显趋势，说明相关性真实；如果是随机散布，果断忽略。

### 7.4 注意事项

1. **数据延迟**：龙虎榜数据在收盘后发布（通常 15:30–16:00），盘中只能用前一日的数据
2. **最少数据量**：CR 趋势需要 ≥10 天、聪明钱需要 ≥8 天重叠、叛变检测需要 ≥3 天
3. **节假日陷阱**：非交易日无数据，分析时注意跳过
4. **换月影响**：合约换月时持仓数据可能出现断崖式变化，需人工判断
5. **相关性 ≠ 因果性**：聪明钱的高相关性可能是因为该席位本身就是大玩家、其行为影响价格，而非"预测"价格
